# UI Design System

**Source**: Anthropic (`anthropics/skills` → `frontend-design`)  
**Installs**: 430.6K  
**Install Command**: `npx skilladd anthropics/skills --skill frontend-design`

## Purpose
Official design guidelines from Anthropic (Claude's creators) for **consistent, professional UI** across web (admin) and mobile (Expo) interfaces.

## Your Role
Instead of saying "make it look nice," Claude knows exactly what "professional" means — spacing, typography, colors, component patterns, accessibility standards.

## Real-World Example
- **Web Admin**: Dashboard for garage management — clean tables, proper spacing, accessible dropdowns
- **Mobile**: Conductor app screens — readable text, touch-friendly buttons, visual hierarchy

## What It Covers
- Typography (font sizes, weights, line heights)
- Color system & contrast
- Spacing & layout grids
- Component patterns (buttons, cards, forms)
- Accessibility (WCAG standards)
- Responsive design

## Integration with CarLink
- Guides both `apps/web/app/admin/*` pages
- Ensures Expo mobile screens feel native + consistent
- Prevents design drift between platforms

## When to Use
Every screen design. It's your "taste maker" for professional UIs.
